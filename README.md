# 📸 慈天相片影展 (3D 線上展廳)

這是一個專為學校活動設計的「自動化 3D 相片展廳」。
具備精美的 3D 輪播介面、背景音樂播放，並內建自動化腳本，能一鍵處理海量照片，實現「本地端看高畫質原圖、線上版看高速縮圖」的智慧切換功能。

## ✨ 核心特色

* 🤖 **極簡全自動更新**：只需把照片資料夾丟進去，執行腳本就會自動生成縮圖與網頁資料庫。
* 🚀 **智慧環境偵測**：
    * **本地端 (學校電腦)**：自動讀取 `images/` 裡的高畫質原圖。
    * **線上端 (GitHub Pages)**：自動讀取 `thumbs/` 裡的輕量化縮圖，節省流量且載入極快。
* 🎵 **沉浸式體驗**：內建背景音樂播放器、支援全螢幕模式。
* 📅 **智慧分類導航**：自動根據資料夾名稱（如 `20260325 早會頒獎`）萃取月份，生成月份導航列。
* 🔍 **細節檢視**：支援點擊照片放大觀看，放大時會自動暫停輪播。

## 📂 專案檔案結構

```text
📁 TWSCPS-gallery/
│
├── 📁 images/       # ➡️ [請把新照片放在這裡] 存放高畫質原圖 (不會上傳到 GitHub)
├── 📁 thumbs/       # 系統自動生成的縮圖 (會上傳到 GitHub 供線上版使用)
├── 📁 audio/        # 存放背景音樂 (支援 .mp3)
│
├── 📄 index.html    # 網頁主體 (前端 UI 與邏輯)
├── 📄 build.js      # 自動化核心腳本 (用來壓縮圖片與生成 data.js)
├── 📄 data.js       # 系統自動生成的資料庫 (給 index.html 讀取的清單)
│
├── 📄 package.json  # 紀錄依賴套件 (如 sharp)
└── 📄 .gitignore    # 阻擋大檔案 (原圖與 node_modules) 上傳的防護罩
